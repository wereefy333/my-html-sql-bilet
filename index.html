Билет 1

Вопрос

Создайте таблицу "Студенты" с полями: id (первичный ключ), имя, фамилия, курс.
Вывести всех студентов 2-го курса.

Ответ

CREATE TABLE Студенты (
    id INT PRIMARY KEY,
    имя VARCHAR(50),
    фамилия VARCHAR(50),
    курс INT
);

SELECT * FROM Студенты WHERE курс = 2;

Билет 13

Вопрос

Создайте таблицы "Клиенты" (id, имя) и "Заказы" (id, сумма, id_клиента).
Вывести общую сумму заказов каждого клиента.

Ответ

CREATE TABLE Клиенты (
    id INT PRIMARY KEY,
    имя VARCHAR(100)
);
CREATE TABLE Заказы (
    id INT PRIMARY KEY,
    сумма DECIMAL(10,2),
    id_клиента INT,
    FOREIGN KEY (id_клиента) REFERENCES Клиенты(id)
);

SELECT Клиенты.имя, SUM(Заказы.сумма) AS Общая_сумма
FROM Клиенты
LEFT JOIN Заказы ON Клиенты.id = Заказы.id_клиента
GROUP BY Клиенты.имя;

Билет 21

Вопрос

Создайте таблицы "Категории" (id, название) и "Товары" (id, название, id_категории).
Вывести количество товаров в каждой категории.

Ответ

CREATE TABLE Категории (
    id INT PRIMARY KEY,
    название VARCHAR(100)
);
CREATE TABLE Товары (
    id INT PRIMARY KEY,
    название VARCHAR(100),
    id_категории INT,
    FOREIGN KEY (id_категории) REFERENCES Категории(id)
);

SELECT Категории.название, COUNT(Товары.id) AS Количество_товаров
FROM Категории
LEFT JOIN Товары ON Категории.id = Товары.id_категории
GROUP BY Категории.название;
